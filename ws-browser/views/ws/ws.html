What's new in v1.0?
  • Add/edit workspace descriptions
  • Copy, delete, and rename objects
  • Filter by object types
  • The "View HTML" has been replaced with direct links to KBase "Landing Pages".
    Object details and download can be found by hovering over an object and selecting "more".
  • View object history [rollback coming soon]
  • View deleted objects [undelete coming soon]
  • Access workspaces via url. Ex: Click Me
  • Workspaces are now stored as username:workspace_name by default when creating a workspace in the WB

Looking for the old Workspace Browser?
The old Workspace Browser is still temporarily available here, but is no longer maintained and KBase services no longer use these workspaces. All of the data from the old Workspace Service is safe and is in the process of being transferred to new WS. The old Workspace Browser will go offline in ~2 months.
